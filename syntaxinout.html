語法

Python使用縮排代表程式區塊範圍，相同縮排距離的程式會被視為相同程式範圍區塊。以下範例使用條件if，if區塊程式範圍必須縮排並對齊，else區塊也一樣。

if (2==2):
    print("yes")
else:
    print("no")

輸入

程式藉由input()函數取得輸入值，input()用於取得到換行 Enter 之前的值，並將資料存成字串。

a=input()        
print(a)

如果需要將輸入值進行數學運算，需要將輸入文字轉成數字類型才能進行運算，函數int()與float()將括號中內容轉成整數與具小數數字。

a=input()        
print(int(a)+3)
print(float(a)+1.5)

在E-Tutor程式資料輸入是藉由讀入資料檔案，當E-Tutor程式資料輸入為每行1個時，請使用第一個程式區來讀取輸入資料，第一個程式區塊每次將輸入值藉由input()讀入變數l，如果一行包含許多輸入值時，我們會將input()字串藉由split()函數將字串依照split()函數中的分割字元進行分割，分割完後會傳回字串清單，當程式讀到輸入檔案最後(EOF)時，會產生EOFError，因使程式會離開迴圈while True。

while True:        
    try:        
        l= input()             
    except (EOFError):        
        break

while True:        
    try:        
        l= input().split(' ')  //l[0]儲存第一行第一個輸入值, l[1]儲存第一行第二個輸入值           
    except (EOFError):        
        break

輸出

Python使用print()函數執行輸出，如果使用者要執行格式化輸出，需指定輸出格式如下列程式範例。print中輸出內容中{}指定對應至format的輸出值，其中{:2d}代表輸出至少2位整數。{:5.2f}代表輸出具小數數字，總長度最小為5位，而且小數2位。最後{:>5}代表輸出最少5字，同時文字靠右擺放，如果文字要靠左擺放則使用<。請注意如果輸出中需要跳行則引號輸出終須加上 \n 。

print('Number 1:{:2d},Number 2:{:5.2f},String:{:>5}'.format(1,2,'abc'))

練習題

E-Tutor基本題目專區第2。第2題為英哩轉公里，每一行只有一個數字，將數字讀入後乘以1.6，並印出小數1位的結果。執行程式請點選題目中提交(新judge)，之後在左邊程式碼塊中輸入程式，之後請選擇程式語言為Python，版本請選擇3.6.9，最後請點選提交。提交後，請點選查看結果，應該會看到5組測試資料，其結果顯示正確。程式如下，每行一個數字，因此採用input()，由於input()讀入會被儲存為字串，因此透過轉換函數float()將文字性數字轉換為具小數點類型的數字，之後藉由輸出print()將答案印出，請注意：因為答案要求輸出1位小數，因此使用輸出格式{:.1f}來設定輸出格式。

while True:        
    try:        
        l= input() //讀入每行資料
        print('{:.1f}'.format(float(l)*1.6))  //將文字性數字轉為實數並乘以1.6後，再以1位小數格式印出     
    except (EOFError):        
        break
