條件

當使用者需要依據條件來執行對應的動作時，我們就必須使用條件，請注意不管是條件、迴圈或是函數，在命令之後都必須加上 : 代表該命令區塊開始，在 : 下有相同縮排位置的命令則屬於該區塊，Python條件式包含如下：

if (2==2): print("2=2")  //僅包含條件對的動作，if條件之後必須加 :

if (2>2):
    print("2>2")  //包含條件對的動作，if條件之後必須加 :
elif (2==2):
    print("2=2")  //當elif之前的條件都不成立時的動作，elif之後必須加 :

if (2>2):
    print("2>2")  //包含條件對的動作，if條件之後必須加 :
elif (2!=2):
    print("2!=2")
else:
    print("2=2")  //包含條件錯的動作，else之後必須加 :

迴圈

當程式重複執行某些動作可使用迴圈命令，Python迴圈命令包含 for 與 while。迴圈參考資料

for迴圈用法下列範例，使用方法為 for 變數 in 清單: ，他會將清單變數中的每各項目都執行依次，for迴圈最常搭配的清單函數為 range，range函數常用使用方式為 range(數字)、range(起始值,終止值)與range(起始值,終止值,間距值)。請注意range函數產生值不包含終止值。

print(range(10)) //印出1,2,3..,9
print(range(5,10))  //印出5,6,7,8,9      
print(range(1,10,2))  //印出1,3,5,7,9

a=0
for i in range(1,10):  //i值從1到9      
    a=a+i  //將i加入a
print(a)

a=0
i=1
while (i<10):  //i值從1到9      
    a=a+i  //將i加入a
    i=i+1
print(a)
